# Art Institute of Chicago DataTable

A React application that displays artwork data with server-side pagination and persistent row selection.

## Features

- **Server-Side Pagination**: Fetches only the current page data from the API
- **Persistent Row Selection**: Selections persist when navigating between pages
- **Custom Selection Panel**: Select first N rows with input field
- **No Pre-fetching**: Does NOT fetch other pages for selection (meets requirements)
- **TypeScript**: Full type safety
- **Vite**: Fast build tool
- **PrimeReact**: Professional UI components
- **Tailwind CSS**: Utility-first styling
